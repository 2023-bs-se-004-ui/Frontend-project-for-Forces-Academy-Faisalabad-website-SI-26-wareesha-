🔧 Customization Tips
Change Colors
Edit the CSS in css/style.css:

css
#mainNav {
    background: #0b1a3b;      /* Dark Navy Blue */
}

.btn-warning {
    background: #ffc107;      /* Yellow/Gold */
    color: #0b1a3b;
}

.text-warning {
    color: #ffc107 !important;
}
Change Hero Background
Replace the gradient in css/style.css:

css
#hero {
    background: linear-gradient(135deg, #0a1f44 0%, #1a4a6b 100%);
    /* OR use an image: */
    /* background-image: url('../images/hero-bg.jpg'); */
}
Add Your Own Logo
Replace the text logo in the navbar:

html
<a class="navbar-brand" href="index.html">
    <img src="images/logo.png" alt="Forces Academy" height="40">
</a>
Add More Nav Links
Add new items to the navigation:

html
<li class="nav-item">
    <a class="nav-link" href="newpage.html">New Page</a>
</li>
